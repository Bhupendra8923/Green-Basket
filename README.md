## ✨ Features

### 🔍 Smart Search
- Live search dropdown as you type
- Click any result → full **Product Detail Modal** opens
- Highlights matching text in results
- Filters product grid simultaneously

### 🛍️ Product Detail Modal
- Large product display with badge labels (SALE / NEW / HOT / ORGANIC)
- Quantity selector before adding to cart
- Add to Cart with live in-cart count update
- Wishlist toggle ❤️ / 🤍
- Nutrition / Info panel
- Related products from same category

### 🛒 Shopping Cart
- Slide-in cart sidebar
- Quantity controls (+ / −) per item
- Coupon codes: `FRESH10`, `GREEN20`
- Auto free delivery above ₹499
- Tax calculation & order summary
- Checkout confirmation

### ❤️ Wishlist
- Save products from card or detail modal
- Add to cart directly from wishlist
- Badge count in header

### 🎛️ Filters & Sorting
- Sort by: Price, Rating, Name A–Z, Best Discount
- Price range slider
- Minimum star rating filter
- Label filters: Organic / On Sale / New Arrivals
- Clear all filters button

## 🚀 Getting Started

### Option 1 — Open directly
```bash
# Just open the file in your browser
open index.html
```

### Option 2 — Clone & run
```bash
git clone https://github.com/YOUR_USERNAME/greenbasket.git
cd greenbasket
open index.html
```

### Option 3 — Deploy on GitHub Pages
1. Go to your repo → **Settings** → **Pages**
2. Source: `Deploy from a branch`
3. Branch: `main` / `root`
4. Click **Save** → your site is live at:
   `https://YOUR_USERNAME.github.io/greenbasket/`
